# SeminaireDiotSciaci2026

Projet client lourd .NET moderne vide (WPF, .NET 8, nullable et implicit usings activés).

## Structure

- `src/SeminaireDiotSciaci2026.App/`

## Prerequis

Installer le SDK .NET 8 (ou plus recent):

- https://dotnet.microsoft.com/download

## Commandes utiles

Depuis la racine du depot:

```powershell
dotnet restore src/SeminaireDiotSciaci2026.App/SeminaireDiotSciaci2026.App.csproj
dotnet build src/SeminaireDiotSciaci2026.App/SeminaireDiotSciaci2026.App.csproj
dotnet run --project src/SeminaireDiotSciaci2026.App/SeminaireDiotSciaci2026.App.csproj
```
